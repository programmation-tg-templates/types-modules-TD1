# Types et modules - TD1 - Informations sur un film

## Consignes

1. Crée un type appelé `Film` qui représente un film avec les propriétés suivantes :

   - `titre: string`
   - `annee: number`
   - `realisateur: string`
   - `disponible: boolean`

2. Crée une variable `film1` de type `Film` et donne-lui une valeur (par exemple : Titanic).
3. Crée une fonction `afficherFilm(f: Film): void` qui affiche ses informations dans la console de cette manière :
   ```
   Titanic (1997) - Réalisé par James Cameron
   Disponible : oui
   ```
4. Appelle la fonction `afficherFilm` avec l'objet `film1`.
